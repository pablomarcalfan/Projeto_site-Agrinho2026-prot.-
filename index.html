Tecagro é um site voltado aos agricultores, para que construam um futuro lucrativo e sustentável

Para que a agricultura seja sustentável, primeiro precisamos combater a desinformação no setor agropecuário.

Alguns mitos comuns incluem:

  • O agronegócio é o principal destruidor da Amazônia
  • O Brasil usa hormônios para criar frangos
  • Gasta-se 15.000 litros de água para produzir 1 quilo de carne
